# 校园表白墙 💕

一个简单的校园表白墙应用，支持文字表白、图片和视频上传功能。

## 功能特点

- ✨ 支持文字表白
- 🖼️ 支持图片上传（JPG、PNG、GIF）
- 🎥 支持视频上传（MP4、WebM）
- 📱 响应式设计，适配移动端
- 🔄 拖拽上传功能
- 🎨 多种表白类型（爱情、友情、欣赏、感谢）
- 🔐 后台管理功能

## 项目结构

```
表白墙/
├── index.html          # 主页面
├── admin.html          # 管理页面
├── admin-dashboard.html # 管理仪表板
├── style.css           # 样式文件
├── script.js           # 主脚本文件
├── admin-script.js      # 管理脚本
└── README.md           # 说明文档
```

## 本地运行

1. 下载项目文件
2. 用浏览器打开 `index.html`
3. 开始使用表白墙功能

## 部署到GitHub Pages

1. 在GitHub创建新仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择 `master` 分支作为源
5. 访问生成的页面链接

## 技术栈

- HTML5
